
# CCCUP-XL

Website CCCUP XL(nyicil)
## [🕹️] Start Contributing

Start new python venv
```bash
  python -m venv .venv
  .venv\Scripts\activate
```

Install dependencies
```bash
  pip install -r requirements.txt
```

Starting Django
```bash
  cd cccup
  python manage.py runserver
```

it runs on https://localhost:8000
